# Healthcare Data Analysis & Risk Insights

## Project Overview
This project analyzes healthcare patient data to identify stroke risk factors
and uncover insights that support preventive healthcare decisions.

## Dataset
- Source: Kaggle – Stroke Prediction Dataset
- Records: ~5,100 patient records
- Features: Age, gender, hypertension, heart disease, BMI, glucose level, smoking status, stroke

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL (MySQL / PostgreSQL)
- Excel
- Power BI
